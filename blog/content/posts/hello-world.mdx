## 为什么要有这个博客

## 代码示例

```python
# AI 前沿日报 API 调用示例
import requests

def fetch_daily_report():
    resp = requests.get(
        "https://aihot.virxact.com/api/public/daily",
        headers={"User-Agent": "Mozilla/5.0 ..."}
    )
    return resp.json() if resp.ok else None

data = fetch_daily_report()
print(f"今日 {sum(len(s['items']) for s in data['sections'])} 条精选")
```
